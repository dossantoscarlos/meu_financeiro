# Código de Conduta

Este projeto adota um padrão de conduta para garantir um ambiente
respeitoso, produtivo e profissional para todos os participantes.

Ao contribuir, você concorda em seguir estas regras.

---

## Comportamentos esperados

- Comunicação respeitosa e objetiva
- Críticas focadas no código, não na pessoa
- Abertura a feedback técnico
- Colaboração de boa fé
- Respeito a diferentes níveis de experiência

---

## Comportamentos inaceitáveis

- Ataques pessoais, insultos ou linguagem ofensiva
- Assédio, intimidação ou discriminação
- Provocações deliberadas ou flame wars
- Spam, trolling ou ruído recorrente
- Qualquer comportamento que prejudique a comunidade ou o projeto

---

## Escopo

Este Código de Conduta se aplica a:

- Issues
- Pull Requests
- Comentários em código
- Discussões
- Qualquer interação relacionada ao projeto

---

## Aplicação

Os maintainers do projeto são responsáveis por interpretar e aplicar
este Código de Conduta.

Ações possíveis incluem:

- Advertência
- Fechamento de issues ou PRs
- Bloqueio temporário ou permanente de contribuições
- Remoção de acesso ao repositório

Não há obrigação de aviso prévio.

---

## Relato de violações

Se você presenciar ou sofrer uma violação deste Código de Conduta,
entre em contato com os maintainers do projeto de forma privada.

Relatos serão avaliados com seriedade e confidencialidade.

---

## Considerações finais

Este Código de Conduta existe para proteger o projeto e seus
colaboradores, não para limitar discussões técnicas.

Discordância é normal. Desrespeito não.
